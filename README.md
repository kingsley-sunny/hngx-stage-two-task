# hngx-stage-two-task (Backend Task)

## Installation

Follow these steps to install the project on your machine:

Clone the repo:

```bash
https://github.com/kingsley-sunny/hngx-stage-two-task.git
cd hngx-stage-two-task
```

Install the dependencies:

```bash
yarn
```

Running locally:

```bash
yarn start
```
